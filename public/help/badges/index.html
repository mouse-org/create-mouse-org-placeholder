Help: Badges and Credly.com

On Mouse Create users age 13 and older can be awarded digital badges issued through a third party site, credly.com. To be eligible to receive badges users need to either create a Credly account through Mouse Create, or connect an existing Credly account. Users can create a Credly account or connect to Credly through their profile page.

When projects associated with a badge are added to a group’s playlist the badge image will display on the project card in the playlist. As users submit projects for review they will appear to Educators in a table of submitted projects on the group page.

Once each of the following steps has happened users will be eligible to be awarded a badge:

  • User is age 13 or older and has connected to Credly
  • User has submitted or been tagged as a collaborator on every project associated with the badge to be reviewed
  • Educator has reviewed every project associated with the badge, and marked the project as reviewed in the Project Tracking section of the group page

Once a user is eligible to be awarded a badge, the badge and their name will be listed in a Badges table in Project Tracking. To award the badge an Educator will click on the name of the badge, agree to terms and click “Award Badge". This will award the badge through the user’s Credly account, and create a publicly shareable URL with the user’s project work shown. Once the badge has been awarded it will display on the user’s Mouse Create profile and the user will be able to accept and share the badge on Credly.

The Badge table will not appear on the group page until at least one user is eligible to be awarded a badge.
