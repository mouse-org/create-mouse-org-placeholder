Using Scratch 3.0

This course features several embedded games made with Scratch, which currently only supports touchscreen with the following workaround:
Open a browser tab to https://beta.scratch.mit.edu/.

When you see the "Welcome to Scratch 3 Preview" modal click "View Scratch 2.0 Project."

Copy and paste the URL of the game you wish to play into the modal. Play the game. Touching the screen is read by scratch as a mouse click. On January 2, 2019 Scratch 3 will be officially released and this workaround will no longer be required to play these games on a touchscreen.
